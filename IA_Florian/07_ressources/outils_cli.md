## Boucle shell avec affichage formaté par fichier
📅 2025-05-22

```bash
find . -type f | while IFS= read -r fichier; do
  echo
  echo "╭────────────────────────────"
  echo "│ 📄 Fichier : $fichier"
  echo "╰────────────────────────────"
  cat -n "$fichier"
done
```
