## Redirection silencieuse de script BAT avec VBS
📅 2025-05-13

```vbs
Set WshShell = CreateObject("WScript.Shell")
WshShell.Run chr(34) & "script.bat" & Chr(34), 0
Set WshShell = Nothing
```
